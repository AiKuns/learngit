初始化一个Git仓库
  git init   
添加文件到Git仓库
  使用命令git add <file>；
  使用命令git commit -m "<message>"，完成。
查看仓库的当前状态
  git status
查看文件与上一个版本的difference
  git diff <file> 查看工作区和暂存区差异
  git diff --cached <file> 查看暂存区和仓库差异
  git diff HEAD <file> 查看工作区和仓库的差异
查看当前版本的操作记录
  git log
查看所有版本的操作记录
  git reflog
回退版本
  git reset --hard HEAD^
  git reset --hard <版本ID>
